Listener JS

Biblioteca padrão (já inclusa no layout base) que trata eventos de teclado (keydown) possibilitando adicionar atalhos, além de oferecer suporte a tabulação na tecla Enter para formulários.

<script src="{% static 'js/listener.js' %}"></script>

Lógica utiliza de dicionário (object) pré carregado [ SHORTCUT_MAP ] onde a key representa a combinação de atalho a ser capturada e o value o comando a ser executado.

A Key deve ser formada pela tecla alvo seguido das letras T ou F (true ou false) que representam a combinação com as teclas Alt, Ctrl e Shift (respectivamente), ou seja a key 'mFTF' responde pelo comando Ctrl + M (case insensitive).

No value deve ser inserido a função que responde pelo atalho, Ex.: SHORTCUT_MAP['dTFF'] = () => {document.getElementById('id_el').click()}

Por padrão já são carregados os atalhos abaixo no dicionario SHORTCUT_MAP:

SHORTCUT_MAP = {
  'vTFF':{...}, // Alt + V: Evento click do elemento ID back
  'nTFF':{...}, // Alt + N: Evento click do elemento ID add
  'lTFF':{...}, // Alt + L: Evento click do elemento ID clear
  'gTFF':{...}, // Alt + G: Evento click do elemento ID submit
  '/FTF':{...}, // Ctrl + /: Evento click do elemento ID search
  'dTFF':{...}, // Ctrl + D: Aciona evento click do elemento ID download
  'iTFF':{...}, // Alt + I: Aciona evento click do elemento ID home
  '.TFF':{...}, // Alt + .: Aciona evento click do elemento ID app_root
  'mTFF':{...}, // Alt + M: Aciona evento click do elemento ID messages
  'sTFF':{...}, // Alt + S: Aciona evento click do elemento ID system
  'dTFF':{...}  // Ctrl + D: Aciona evento click do elemento ID docs
}
Inserir novo atalho (ou alterar):
SHORTCUT_MAP['xFTF'] = () => {...};
Tabulando forms fields ao pressionar Enter

Listener.js implementa a tabulação de campos de formulário ao pressionar a tecla Enter, para ativar crie uma variavel TAB_ON_ENTER e defina o valor dela para true

var TAB_ON_ENTER = true;
Pressione Enter para tabular

Lógica somente faz a tabulação de campos dentro de um elemento form .

Implementação trata campos desabilitados (disabled) e campos não visiveis, e busca próximo elemento disponível.

Para evitar tabulação em um input em especifico adicione o atributo listener-js="escape-tab"

I18N JS Internationalization

Implementa funcionalidade para tradução de texto para outros idiomas (pre cadastro de entradas de chaves para cada idioma).

Como funciona:
  1. Ao carregar página lib verifica qual o language do cliente através do navigator.language.
  2. Caso linguagem seja diferente da default pt-BR será solicitado ao servidor (via ajax) arquivo com biblioteca para o requerido idioma.
  3. Caso linguagem seja a default, nenhum código é realizado.
  4. No HTML defina o atributo data-i18n com o valor da chave a ser procurada
    Ex: <span data-i18n="core.user.new">Novo Usuário</span>
  5. Será buscado na extrutura do objeto caminho informado no atributo data-i18n, caso encontrado retorna valor da chave
  6. Caso não encontrado chave exibe alert no log do navegador console.log() com a chave(s) não encontrada
  7. Por padrão, o texto traduzido será inserido no innerHTML do elemento, este comportamento pode ser alterado definido data-i18n-targetEx: <input type="text" data-i18n="core.user.name" data-i18n-taget="placeholder" />
Importe a biblioteca ao final do body (requer dot.js): 
<script src="{% static 'js/i18n.js' %}"></script>
No html: 
<span data-i18n="common.submit">Salvar</span>
Exemplo de arquivo de chaves i18n__en-US.json: 
{
    "common":{
        "register": "Register",
        "cancel": "Cancel",
        "submit": "Save",
        ...
    },
    "core":{
        "user":{
            "created": "User Created",
            ..
        }
    }
}

URL JS

Funções para manipulação da URL (URI) no método GET.

urlFilters(filter, value): Faz upate na página atual adicionando parametro na url
urlFilter('empresa', '5')
urlFilters(filters, values): Mesmo conceito de urlFilter, porém pode receber multiplos parametros
urlFilters(['empresa', 'filial'], ['5','2'])
urlFilterTo(url, filter, value): Muda para nova url e adiciona filtro
urlFilterTo('dot.com.br/usuarios', 'nome', 'rafael')
urlFilterToogle(filter, values): Altera o valor do filtro baseado em sequencia predefinida
urlFilterToogle('ativo', ['true', 'false'])
urlFiltersClean(filters): Limpa filtros da URI
filtersClean(['nome', 'email'])
urlRedirect(url): Chama nova URL passando todos os parâmetros atuais na requisição
urlRedirect('dot.com/usuarios_bloqueados')
urlHasParam(param): Verifica se o parâmetro informado consta na URI
if(urlHasParam('nome')){...}

// FONE MASK
function inputHandler(masks, max, event) {
  var c = event.target;
  var v = c.value.replace(/\D/g, '');
  var m = c.value.length > max ? 1 : 0;
  VMasker(c).unMask();
  VMasker(c).maskPattern(masks[m]);
  c.value = VMasker.toPattern(v, masks[m]);
}
var telMask = ['(99) 9999-99999', '(99) 99999-9999'];
var fone1 = document.getElementById('id_fone1');
var fone2 = document.getElementById('id_fone2');
VMasker(fone1).maskPattern(telMask[0]);
VMasker(fone2).maskPattern(telMask[0]);
fone1.addEventListener('input', inputHandler.bind(undefined, telMask, 14), false);
fone2.addEventListener('input', inputHandler.bind(undefined, telMask, 14), false);

fone1.dispatchEvent(new Event("input"));
fone2.dispatchEvent(new Event("input"));
