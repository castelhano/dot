Instalação
Atenção! Todo o processo de instalação descrito neste artigo foi realizado num ambiente linux (Ubuntu 20.10), pequenas alterações podem ser necessárias para outros ambientes

Certifique-se que o sistema operacional está devidamente atualizado

sudo apt-get update && sudo apt-get upgrade

Sistema requer python na versão 3.6 ou superior, execute o comando abaixo para verificar qual a versão do python instalada

python3 --version

Opcional: Configurando um ambiente virtual na pasta do projeto.

python3 -m venv ./venv
Para ativar o ambiente virtual digite: source venv/bin/activate

Utilizamos o gerenciar de aplicações do python para instalar o framework django e a biblioteca pillow usada para manipulação de imagens

pip install django pillow

Utilizamos o sistema open source de versionamento git, e o repositório github para distribuição do software

sudo apt-get install git

Clone o projeto para a máquina local

git clone https://github.com/castelhano/dot.git
Atenção! É necessário ter uma conta no github com acesso ao repositório da aplicação, entre em contato com o adminsitrador do sistema caso tenha problemas nesta etapa

A aplicação utiliza por padrão o sistema de banco de dados sqlite3, que tem exelente desempenho e é de facil manutenção. Porém o django oferece suporte aos principais bancos de dados disponiveis, caso queira utilizar outro banco de dados vá até a parte de configuração do sistema aqui.

O próximo comando cria o banco de dados, tabelas, indices e demais componentes para o BD definido.

python manage.py migrate
